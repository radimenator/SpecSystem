# SpecSystem

## Role projektu
SpecSystem je kanonické uložiště specifikací projektů.

## Vztahy
- kanonická specifikace: `projects/spec-system/`
- zdrojová evidence: `shared/sources/spec-system/`
- ArchiMate model: `shared/models/archimate/`
- výstupy: `projects/spec-system/outputs/`

## Klíčová témata
- specifikace jako source of truth
- oddělení poznámek od specifikace
- Source Evidence Repository
- Obsidian / LLM Wiki
- ArchiMate Model Repository
- Output / Document Generation
- local-first + VPS orchestrace
