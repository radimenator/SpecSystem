# Summary

Stručné shrnutí zdroje: SpecSystem je kanonické uložiště specifikací projektů. Wiki funguje jako znalostní vrstva pro tato uložiště a ArchiMate model poskytuje formální projekci architektury.

# Concepts

- **specifikace**: Detailné popisy a požadavky na určitý projekt nebo systém.
- **wiki**: Digitální prostředek pro sdílení a správu společných informací mezi týmem.
- ** znalostní vrstva**: Systém organizace a struktury znalostí, které slouží k podpoře procesů vytváření a používání informací.
- **ArchiMate model**: Formální jazyk pro vizualizaci a popis architektury systémů a orgánů.

# Entities

- **SpecSystem**: Uložiště specifikací projektů, kanonická reference na specifikace pro určité projekty.
- **Wiki**: Slouží jako znalostní vrstva pro SpecSystem.
- **ArchiMate model**: Formální popis architektury systémů a orgánů vztahující se k SpecSystem.
