# Summary

Dokument popisuje hlavní oddělení systému SpecSystem, které zahrnuje různé nástroje a koncepty pro projektovou práci, správu zdrojových dokumentů, znalostní vrstvy, kanonickou specifikaci a výstupy.

# Key Concepts

- znalostní vrstva: interpretace a syntézy zdrojových dokumentů
- source of truth: kanonická specifikace v systému SpecSystem

# Entities

- Roam Research: nástroj pro projektový kontext a myšlení
- Source Evidence Repository: sbírka zdrojových dokumentů a modelů
- Obsidian / LLM Wiki: nástroj pro znalostní vrstvu
- SpecSystem: systém poskytující kanonickou specifikaci
- ArchiMate Model Repository: úložiště pro formální architektonický graf

# Architecture Facts

- Systém SpecSystem zahrnuje nástroje pro správu projektů, vytváření dokumentace a generování formálních specifikací.
- Znalostní vrstva je interpretována pomocí Obsidianu a LLM Wiki.

# Decisions / Constraints

- SpecSystem funguje jako kanonický zdroj pravdy pro systém.
