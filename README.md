# SpecSystem

SpecSystem je kanonické uložiště specifikací projektů.

## Oddělení odpovědností
- ~/Projects = implementace a kód
- ~/SpecSystem = specifikace
- Roam Research = projektový kontext, poznámky, aktivity

## Základní pravidla
- každá aplikace má jeden root specifikace
- root specifikace je uložen v ~/SpecSystem/projects/<project-id>
- specifikace je kanonický zdroj pravdy
- významné změny se zapisují do historie a Git logu

## Minimální struktura projektu
- 00-meta
- 10-motivation
- 20-scope
- 30-architecture
- 40-components
- 50-decisions
- 60-links
- 70-regeneration
- 80-history
